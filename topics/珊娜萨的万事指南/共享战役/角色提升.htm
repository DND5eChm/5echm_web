角色提升 Character Advancement
在共享战役中，玩家不以经验值为升级的条件，改而使用经验点来升级。这一系统将激励每一个角色（与其玩家）参与到一场游戏的进程中去。

一个角色在冒险者将根据其进行的冒险设计的时间获取经验点，以1小时=1经验点的方式换算。请注意这里的奖励是基于冒险的设计游玩时间，而非实际在桌上游玩的时间。这一奖励在冒险完成时才生效。打个比方，如果你完成了一场设计时2小时的冒险，你将获得2点经验点，即使你的团中花费了超过2个小时去游玩完成它也是2点。

此外，如果一个角色完成了一场为更高 阶层Tier 设计的冒险，那个角色将可以额外获得1点经验点。举例来说，如果一个2级角色完成了一场为6级设计的冒险（设计时2小时），这个角色将获得3点经验点。

根据游戏时间来衡量经验奖励似乎的一种奇怪的方式，但这一概念与共享战役的运作方式是一致的。若某个角色玩了10个小时，它与其他10小时角色获得的经验点数量是相同的，无论该角色是在讨伐巨龙，还是一直蛰伏在酒馆里窃取情报。这种方法可以确保玩家喜欢的风格既不会受到惩罚，也不会得到奖励。无论玩家是专注于角色扮演和社交互动，还是在战斗中击败怪物，抑或是寻找巧妙的方法避免战斗，该系统都会给予角色相应的奖励。

使用经验点 Using Checkpoints
获得下一级所需的经验点数量取决于升级前角色的等级：

  • 在1~4级时，你只需要4点经验点就足以去提升至下一级了。
  • 在5级或更高等级时，你需要8点经验点才能提升至下一级。

当一次游戏进程结束时，如果角色有足够升级的经验点，那角色必须得在此升级。而升级后多出来的经验点可被用于下次有升级机会的时候使用。
