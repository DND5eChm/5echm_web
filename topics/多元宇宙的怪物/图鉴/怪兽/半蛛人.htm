半蛛人 Chitines

半蛛人是侍奉罗丝的、带有蛛形状纲特征的多臂人形生物。大多数都住在幽暗地域的聚落之中，并对抗恶魔蜘蛛女王的敌人。很久以前，罗丝的教会首先通过用精灵囚犯进行可怕的仪式，将他们转变为具有人形和蜘蛛特征的生物，她们的创造者将之称为“半蛛人”。黑暗精灵的意图是首先创造出为教会服务的忠诚战士，然后通过与他们的联系为罗丝服务。这位女神并不接受这样的安排。
作为惩罚，蛛后扭曲了她的追随者的仪式。仪式过程通常会将试验品转化为她的崇拜者所预想的蜘蛛外形生物，不过偶尔，精灵会被转化半蛛神眷：一种能够自己引导并产下更多半蛛人的蛛形状纲怪物。这些半蛛神眷很快就带领半蛛人反抗并抛弃他们的创造者，在幽暗地域的其他地方建立起自由殖民地。偶尔，也能在偏远阴暗的地表区域发现与罗丝的敌人交战的半蛛人们。
如今，罗丝的教会仍然会在他们需要的时候制造半蛛人。在没有神眷的情况下，半蛛人是很好的劳力，如果教会发现一个独立的半蛛人群体并想要渗入其中，它们就能十分有用。然而，一旦制造的过程中产生了半蛛神眷，教会就会立即摧毁这个生物。
作为罗丝的仆从，半蛛人热爱蜘蛛。她们饲养蜘蛛和类似的蛛形纲动物，比如穴钓者（也在本书中）。半蛛人聚落将罗丝像灯塔一样供奉在神龛内，吸引蜘蛛和其他侍奉她的存在。任何半蛛人建立聚落所在的区域，很快都会成为一个蛛网遍布、阴森和危险之地。
半蛛人与蜘蛛相似，但她们表现得则更像群居性的昆虫，比如蚂蚁。他们分为工人和战士阶级；神眷出现时则占据了聚落阶级的顶层。每一个半蛛人都有一个社会地位，伴随着与之相关的职责，所有的半蛛人都愿意牺牲自己来保护聚落的半蛛神眷。每个半蛛人都有喷丝器官，并缓慢地产生用于建造地板、墙壁、结构、物体和陷阱的网状物，这些都有利于保护群落。一个战士可能要负责制作蛛丝甲（像兽皮或皮革一样坚硬），而一群工人可能会挖掘陷阱，用易碎的带子盖住它，用松散的泥土伪装，使其看起来像一个坚实的表面。

精类血统Fey Ancestry。半蛛人进行对抗魅惑的豁免检定时具有优势，且魔法效应无法使半蛛人陷入睡眠。
日照敏感Sunlight Sensitivity。在日光照射下，半蛛人的攻击检定具有劣势，其依靠视力的感知（察觉）检定也具有劣势。
蛛网感知 Web Sense。与蛛网接触时，半蛛人可以知晓其他与该蛛网接触生物的准确位置。
蛛网行者 Web Walker。半蛛人在蛛网上移动时无视蛛网造成的移动限制。
