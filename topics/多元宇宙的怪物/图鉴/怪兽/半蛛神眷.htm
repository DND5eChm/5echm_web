半蛛神眷 Choldrith

半蛛神眷是直接侍奉罗丝的、巨大蜘蛛形生物。他们统治着半蛛人（在此书中）的聚落，并带领他们参与对抗罗丝与敌人的纷争。
当罗丝的信徒创造第一个半蛛人时，罗丝观看着她的追随者使用奥术魔法与恶魔力量与确保接受仪式者从中幸存所必须的神圣火花。作为对信徒的背叛带来的复仇，蛛后扭曲了创造仪式，使他们有时创造的是半蛛神眷而不是半蛛人。
起初，这些信徒并没有意识到这些他们称为半蛛神眷的新的生物是罗丝对他们不满的迹象。相反，他们十分高兴。因为半蛛神眷可以产下能生出更多半蛛人（以及少部分神眷）的卵，并能在工作中引导半蛛人。但随即，信徒们就意识到了他们的错误——半蛛神眷的身体和灵魂都属于罗丝。半蛛神眷向半蛛人们低语它们对蛛后的崇拜和她们对卓尔的敌意，并带领它们成功地进行了反抗。
半蛛神眷生来就与罗丝有一种神秘的联系，这让它们具备一些神术魔法。它们也构成了大多数半蛛人聚落的统治阶层。一个聚落可以支持大量半蛛神眷，担任指挥官、祭司和检察官等职位。半蛛神眷很少以一种危及整个聚落的方式相互对抗，但它们不断地争夺地位。聚落由一位君主统治，并由君王决定聚落成员执行哪些任务，，包括是否允许半蛛神眷产卵。有时，一个半蛛神眷统治者会从罗丝那里收到一个幻象，激励整个聚落进行一些宏大的，通常是暴力的行动。

精类血统Fey Ancestry。半蛛神眷进行对抗魅惑的豁免检定时具有优势，且魔法效应无法使半蛛人陷入睡眠。
蛛行Spider Climb。半蛛神眷可以攀爬在难以攀爬的表面上，包括倒吊在天花板上，且不需要进行属性检定。
日照敏感Sunlight Sensitivity。在日光照射下，半蛛神眷的攻击检定具有劣势，其依靠视力的感知（察觉）检定也具有劣势。
蛛网感知 Web Sense。与蛛网接触时，半蛛神眷可以知晓其他与该蛛网接触生物的准确位置。
蛛网行者 Web Walker。半蛛神眷在蛛网上移动时无视蛛网造成的移动限制。
