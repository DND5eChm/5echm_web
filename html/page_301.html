魔宠和模板 (Familiar and Template)

 

l  魔宠 (Familiar)

l  进阶魔宠 (Improved Familiar)

l  生物模板 (Creature Templates)

 

 

　　魔宠是一种普通动物，被施法者选中在他的魔法研究中提供协助。它保留它作为动物时的外形、HD、BAB、基本豁免加值、技能和专长，但是在考虑依赖于分类的魔法效果时，它视为魔法兽而非动物。只有一个普通的未受训的动物才能成为魔宠。动物伙伴无法成为魔宠。

　　魔宠会给他的主人带来特殊能力，这些特殊能力只有在主人和魔宠彼此相距1英里内才起作用。

　　在确定魔宠的那些跟主人等级相关的特殊能力时，要将所有具有魔宠职业的等级累加起来。

　　如果魔宠被解约、遗失或者死亡，它能在1周后通过一个消耗 {职业等级×200gp} 的特殊仪式被更换。仪式完成需要8小时。

 

　　魔宠基本说明 (Familiar Basics): 使用魔宠同类生物的数据，但是作如下调整。

　　生命骰：在计算和HD有关的效果时，使用主人的角色等级或者魔宠正常的HD，以高者为准。

　　生命值：魔宠的HP为主人总HP (不包括临时HP) 的一半，和魔宠实际HD无关。

　　攻击：使用主人所有职业等级合计的BAB。用魔宠的敏捷修正或力量修正中较高者加在魔宠的天生武器近战加值上。伤害和魔宠的普通同类生物相同。

　　豁免：在每项豁免检定上，使用魔宠自身的基础豁免加值 (强韧+2，反射+2，意志+0) 或者主人的 (根据主人所有职业等级计算得出)，以较高者为准。魔宠的豁免使用自己的属性修正，也无法从主人那里分享任何其他可能有的豁免加值。

　　技能：在主人或魔宠有受训级数的技能上，使用魔宠同类动物的正常技能级数或者主人的技能级数，以较高者为准。不管哪种情况，魔宠都使用自己的属性修正。但无论魔宠的总技能加值是多少，有些技能依然超过了魔宠的能力而无法使用。魔宠将特技、攀爬、飞行、察觉、隐匿和游泳作为本职技能。

　　魔宠能力说明 (Familiar Ability Descriptions): 所有魔宠都有特殊能力 (或者给它们主人提供能力)，这些能力依赖于主人所有拥有魔宠的职业总等级。上表给出的能力是可以累加的。

　　天生护甲修正 (Natural Armor Adj.): 该数值增强了魔宠的天生防御。

　　智力属性 (Int.): 魔宠的智力值。

　　警觉 (Alertness, Ex): 当魔宠在一臂之距内时，主人获得“警觉”专长。

　　【速查】警觉 (Alertness): 察觉和察言观色检定+2 (若受训10级以上则升为+4)。

　　精通反射闪避 (Improved Evasion, Ex): 有些攻击效果若通过反射豁免可以减少一半伤害。如果魔宠遭到此类攻击，在通过反射检定后它将不受伤害，如果检定失败它也只受到一半伤害。

　　法术共享 (Share Spells, Ex): 法师可以将目标为自身的法术施展在魔宠身上 (如同接触法术) 而不是自己身上。即使某些法术通常不会影响魔宠生物类型 (魔法兽)，主人也可以用这种方式和魔宠共享该法术。

　　情感联结 (Empathic Link, Su): 主人和魔宠之间有着内心上的联结，情感联结的最远距离为1里。主人无法用魔宠的双眼去看，但是他们彼此分享心灵。由于这种连接的天生限制，只有一般的心灵感受可以被传递。因为这种情感联结，主人对那些魔宠曾经遇上的事物、或去过的地点也会建立起相同的关联。

　　传递接触法术 (Deliver Touch Spells, Su): 如果主人达到3级，魔宠可以替他传递接触法术。如果主人施展接触性法术时魔宠和他在一起，他可以指定他的魔宠为“接触者”。魔宠之后即可像主人本人一样去传递此接触性法术。和正常情况一样，如果在法术被传递前主人施展了另外一个法术，该接触法术消散。

　　与主人交谈 (Speak with Master, Ex): 如果主人达到5级，魔宠和主人之间可以通过语言来交流，就像他们正在使用通用语交谈一样。其他生物在没有魔法帮助的情况下无法理解这种交流的内容。

　　与动物同类交谈 (Speak with Animals of Its Kind, Ex): 如果主人达到7级，魔宠就可以和它近似种类的动物进行交流 (包括凶暴动物)。它们之间的交流受对像生物智力的限制。

　　法术抗力 (Spell Resistance, Ex): 如果主人达到11级，魔宠就会获得法术抗力，SR等于 {主人等级+5}。要用法术影响魔宠，施法者必须在施法者等级检定中得到等于或大于魔宠SR的结果值。

　　探知魔宠 (Scry on Familiar, Sp): 如果主人达到13级，每天1次，他可以对魔宠进行探知 (如同施展‘探知’法术)。

 

小型与虫类魔宠 (Small and Vermin Familiars)

 

　　小型体型的魔宠能够如同小型生物一般威胁到自身周围的区域，而且能够用于夹击敌人，不过魔宠和其主人通常都不喜欢使用此类战术，毕竟这么做的结果通常都是魔宠死亡。和超小型或更小的魔宠相比，小体型魔宠也不方便携带在主人身上。要做到这一点通常需要某些形式的魔法物品，比如<次元袋>。将虫类变为魔宠的转化过程会让它获得智力属性，同时移除无心智特性。虫类魔宠可以与自身的主人以及同种虫类沟通，沟通的方式可以是奇怪的动作组合、外观的变化、甚至是些微的气味变化等等。与其他类型的魔宠一样，如果没有借助魔法的力量，别的生物就无法理解这种沟通。

生物模板 (Creature Templates)

 

　　模板是一套成型的规则，用来将一个生物转变为另一种类型。所有的模板都给出了如何将怪物进行调整的准确步骤。

 

　　获得型模板 (Acquired Templates): 有一些模板 (如巫妖模板) 是因为该生物自愿或需要而获得的结果。其他一些模板 (如幽魂模板) 则是由外力强加于某个生物的结果 (比如一个受尽折磨而死的人类会变成人类幽魂)。不过，这两者的共同点在于，模板将生物变成了另一种形式――并非其起始即有的类型。这些模板称为“获得型模板”，它可以在这个生物存在的任何时候添加于其上。

 

　　遗传型模板 (Inherited Templates): 有一些模板，如半龙模板和半炼狱模板，是这个生物出世时即具有的特质。生物带着这个目标降生或被创造出来，并且不可能与它们分开。这些模板被称为“遗传型模板”。

 

简单模板 (Simple Templates)

 

　　简单模板只需要很少的游戏数据调整就能应用在一个生物上，所以使用起来非常简单。举例言之，天界模板或炼狱模板就可以简单地套在用‘召唤怪物 (summon monster)’法术招来的生物身上并立刻用于战斗。

　　所有的简单模板都有两套使用方式――“快速套用”和“重建规则”。快速套用法只需要简单的调整一些生物的投骰来达到效果，而不需要全面翻新怪物的属性和数据――如果是用在召唤生物上将非常合适。重建规则法则较为详细的指出了你需要调整的每一个属性和数据，如果你有足够的时间在开始游戏前捏怪物的话它会很有用。这两个方法制作出来的模板怪物大同小异，只是在属性和能力上略有差异。当然，有一些模板会导致意想不到的情况发生――比如低智力的生物因为进化模板获得了相当的智力，或是虚体生物因为幼体模板而更难在战斗中被命中。

 

进化生物模板 (Advanced Cureature, CR+1)

 

　　进化生物比它们较为普通的同族更为健硕凶猛。

 

　　快速套用：所有检定 (包括伤害检定) 和特殊能力DC +2，AC和CMD +4，每个HD +2HP。

　　重建规则：AC C 天生护甲加值+2；属性 C 所有属性+4 (除非智力属性要求不能高于2)。

 

天界生物模板 (Celestial Creature, CR+0或+1)

 

　　天界生物诞生于上层位面，它们可以通过诸如‘召唤怪物’或‘异界盟友 (planar ally)’法术召唤来助战。如果基础生物拥有5个或更高的HD，那么天界模板的CR调整值为+1。天界生物模板的快速套用和重建规则相同。

 

　　重建规则：感官 C 获得60尺黑暗视觉；防御能力 C 获得如下表所示的能量抗力和伤害减免；法术抗力 C 获得相当于 {新CR+5} 的法术抗力。

巨化生物 (Giant Creature, CR+1)

 

　　拥有巨化模板的生物要比它们的同类更加庞大强壮。超巨型生物无法获得此模板。

 

　　快速套用：所有基于力量和体质属性的检定+2，每HD +2HP，所有基于敏捷属性的检定-1。

　　重建规则：体型 C 增加1个级别；AC C 天生护甲加值+3；攻击能力 C 伤害骰提升1个级别；属性 C 力量和体质获得+4体型加值，敏捷-2。

幼体生物 (Young Creature, CR-1)

 

　　幼体生物是基础生物类型的不成熟版本，这个模板也可以应用来创造要弱于一般水准的怪物。这个模板不可附加于因为成长或进食而升级的生物 (如龙或犬魔) 或是超微型的生物。

 

　　快速套用：所有基于敏捷属性的检定+2，其他所有检定-2，每个HD -2HP。

　　重建规则：体型 C 降低1个级别；AC C 天生护甲加值-2 (最小+0)；攻击能力 C 伤害骰降低1个级别；属性 C 力量和体质-4，敏捷获得+4体型加值。
